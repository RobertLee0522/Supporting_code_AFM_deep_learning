# 優化摘要 - Supporting_code_AFM_deep_learning.py

## 新增功能

### 1. 訓練進度條
- 使用 `tqdm` 函式庫為所有數據生成和預處理循環添加進度條
- 包括球形、立方體、圓柱形粒子生成
- 膨脹（dilation）處理進度條
- 線性偽影添加和邊緣提升處理

### 2. 訓練時的進度和準確率顯示
- **TrainingProgressCallback**: 自定義回調類別
  - 顯示每個 Epoch 的進度
  - 顯示訓練和驗證損失 (Loss/Val Loss)
  - 顯示估算準確率 (Train Acc/Val Acc)
  - 顯示每個 epoch 耗時和預計剩餘時間 (ETA)
  - 每 10 個 epoch 顯示視覺化進度條
  - 訓練結束後顯示總耗時

### 3. 預測準確率評估
新增評估指標：
- **MAE** (Mean Absolute Error): 平均絕對誤差
- **MSE** (Mean Squared Error): 均方誤差
- **RMSE** (Root Mean Squared Error): 均方根誤差
- **PSNR** (Peak Signal-to-Noise Ratio): 峰值信噪比 (dB)
- **SSIM** (Structural Similarity Index): 結構相似性指數
- **Pixel Accuracy**: 像素級準確率 (10% 誤差容限)

## 使用方式

### 安裝依賴
```bash
pip install -r requirements.txt
```

### 執行腳本
```bash
python Supporting_code_AFM_deep_learning.py
```
